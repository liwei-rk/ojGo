## 动态规划矩阵最大和
```
题目描述
现在你面对一个n×m的矩阵，矩阵中的每一个元素都是一个整数，
现在你需要计算从矩阵的左上角走到右下角所走过的所有元素相加的最大和。
注意：只能向右或者向下走，不能走出边界。

解答要求
时间限制：1000ms, 内存限制：64MB
输入
输入第一行包含两个用空格分开的整数n (1≤ n ≤ 100)和m (1≤ m ≤ 100)，
表示n行m列的矩阵；接下来是n行每行包含m个用空格分开的非负的整数A (0 ≤ A ≤ 100)。

输出
输出从矩阵的左上角走到右下角所走过的所有元素相加的最大和。

样例
输入样例 1

2 3
1 2 3
1 0 2
输出样例 1

8

输入样例 2

5 3
14 14 4
76 5 76
78 23 23
45 75 53
52 43 71
输出样例 2

412

提示
Sample test1中最大和为1+2+3+2=8。

Sample test2最大和为14+76+78+45+75+53+71=412
```
```
用二维数组matrix记录输入的矩阵
记f(i, j)表示从左上角走到matrix[i][j]得到的最大和；
怎么走到(i, j)位置？只能是从上面一格下来或左边一格过来，显然f(i, j) = max(f(i-1, j), f(i, j-1)
```
