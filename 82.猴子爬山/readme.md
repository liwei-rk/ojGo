## 递归与动态规划
```
题目描述
一天一只顽猴想去从山脚怕到山顶，途中经过一个有个N个台阶的阶梯，
但是这猴子有一个习惯： 每一次只能跳1步或跳3步，
试问猴子通过通过这个阶梯有多少种不同的跳跃方式？

解答要求
时间限制：1000ms, 内存限制：64MB
输入
输入一组整数，每组只有一个整数N（0<N<=50）

输出
输出对于每组N，有多少种跳跃方式（解决方案数）

样例
输入样例 1

3
50
输出样例 1

2
122106097
```
```
递归与动态规划

设跳上n个阶梯的方式有f(n)种
那么分为两种情况：
第一次跳1阶，总共的方式是f(n-1);
第一次跳3阶，总共的方式是f(n-3)
所以f(n) = f(n-1) + f(n-3)
为了减少重复计算，不用递归，用一个数组从1开始向n计算，即动态规划：
f(1) = 1
f(2) = 1
f(3) = 2
f(4) = f(3) + f(1)
f(5) = f(4) + f(2)
..
f(n)=f(n-1)+f(n-3)
```
